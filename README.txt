MusicUtil is an iOS app that contains a tuner, sight-reading practice, and ear-training practice.

Before building, you must install CocoaPods in the Terminal using:

'sudo gem install cocoapods'

Then navigate to the project directory in the Terminal and run:

'pod install'

To build, open the .xcworkspace file and you can build in one step by simply hitting the trianglar button in the top left labeled "build and then run current scheme".
